# Ultimate Machines
This plugin adds an assortment of useful machines to Minecraft that help automate many different tasks.

## Commands
```/machines get <machine_name>```

## Machines

### Packager
![Packager](http://i.imgur.com/euaRdNS.png)

---
This machine packages items down to their block form.
For example, diamonds are compressed into diamond blocks

#### Crafting Recipe
![Recipe](http://i.imgur.com/PjLgblR.png)

### Unpacker
![Unpacker](http://i.imgur.com/apR20ol.png)

---
This machine does the exact opposite of the Packager: it turns blocks back into their item form

#### Crafting Recipe
![Recipe](http://i.imgur.com/zEUHVW1.png)

### Drainer
![Drainer](http://i.imgur.com/iAfrKAj.png)

---
